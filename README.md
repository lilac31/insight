# Insight - 思考笔记应用

一个仿照 flomo 设计的移动端 web 笔记应用，帮助你快速记录和整理想法。

## ✨ 核心特性

- 📝 **极简输入** - 像发微博一样快速记录想法
- 🏷️ **标签系统** - 使用 `#标签` 快速分类笔记
- 🔍 **智能搜索** - 快速查找历史笔记
- 📊 **可视化统计** - 记录习惯热力图，了解自己的记录规律
- 💾 **本地存储** - 数据保存在浏览器本地，隐私安全
- 📱 **移动优先** - 完美适配手机屏幕，流畅的触摸交互

## 🚀 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 无需安装，无需注册，开箱即用

## 📖 功能说明

### 快速记录
- 在输入框中输入你的想法
- 使用 `#标签名` 为笔记添加分类
- 点击"保存"按钮或按 `Cmd/Ctrl + Enter` 快速保存

### 标签管理
- 系统会自动识别笔记中的 `#标签`
- 点击顶部标签筛选器可查看特定标签下的笔记
- 标签按使用频率排序

### 搜索笔记
- 点击右上角搜索图标
- 输入关键词即可实时搜索
- 搜索结果会高亮显示匹配内容

### 编辑和删除
- 点击任意笔记卡片进入编辑模式
- 可修改内容或删除笔记

### 查看统计
- 点击右上角统计图标
- 查看总笔记数、标签数、今日记录等数据
- 热力图展示最近 49 天的记录情况

## 🎨 设计理念

- **极简主义** - 去除干扰，专注内容
- **快速输入** - 降低记录门槛
- **持续回顾** - 通过标签和搜索重现价值
- **移动优先** - 随时随地记录想法

## 💡 使用技巧

1. **善用标签** - 为笔记添加有意义的标签，方便后续查找
2. **每日回顾** - 定期查看统计，保持记录习惯
3. **简短记录** - 不要纠结完美，先记录下来再说
4. **多场景使用** - 学习、工作、生活都可以记录

## 🔒 隐私说明

所有数据都存储在浏览器的 localStorage 中，不会上传到任何服务器。
清除浏览器数据会同时删除笔记，请注意备份重要内容。

## 🛠️ 技术栈

- 纯原生 HTML/CSS/JavaScript
- 响应式设计
- LocalStorage 数据持久化
- 无需构建工具，直接运行
